# Helpr

**Helpr** is a mobile-first service marketplace that connects **Requesters** (people who need services) with **Skilled Workers** (plumbers, electricians, repair technicians, etc.) in Pakistan.

Users can post jobs, receive bids, chat in real time, and complete jobs with ratings and reviews.

## ✨ Features

### 👤 For Requesters
- Post service requests (plumbing, electrical, repairs, etc.)
- Receive bids from nearby workers
- Chat with workers before hiring
- Track active jobs
- Rate and review completed work

### 🔧 For Workers
- Browse available service requests
- Place bids on jobs
- Chat with clients
- Manage active and past jobs
- Build reputation through ratings

### 💬 Shared Features
- Secure authentication system
- Real-time chat between users
- Worker public profiles with ratings and reviews

---

## 🛠 Tech Stack

- **Mobile App:** Flutter (Dart)
- **Backend:** Node.js + Express.js
- **Database:** Microsoft SQL Server
- **Authentication:** bcrypt password hashing
- **File Uploads:** Multer
- **HTTP Client:** Flutter `http` package

---


## 🚀 About This Project

**Helpr** simplifies hiring local service providers by allowing users to post requests and receive competitive bids from nearby workers.

The platform is designed to increase transparency, reduce cost, and improve trust through ratings and direct communication.
